# 🌻 Girasol Online

Un hermoso dibujo interactivo de un girasol que lleva los gráficos de tortuga de Python a la web utilizando Brython.

<img src="https://github.com/user-attachments/assets/feb9a8e4-be5f-4a32-a814-a2f1f24614b7" width=75% height=75%>

## ✨ Características

- **Animación en Tiempo Real**: Observa cómo florece el girasol paso a paso
- **Control de Velocidad**: Velocidad de dibujo ajustable (1-10)
- **Diseño Responsivo**: Funciona en computadoras de escritorio, tablets y móviles
- **Réplica Exacta**: Reproduce fielmente el resultado original de `sunflower.py`

## 🚀 Demostración en vivo

[Mira la demostración en vivo aquí](https://sunflower-online.netlify.app/)

## 🎮 Cómo usar

1. Ajusta la velocidad que prefieras con el deslizador
2. Haz clic en el botón para comenzar la animación
3. Observa cómo el girasol se dibuja en tiempo real
4. ¡Disfruta el mensaje cuando termine! 😊

## 🛠️ Tecnologías

- **Frontend**: HTML5 Canvas, CSS3, JavaScript
- **Entorno de ejecución de Python**: Brython
- **Gráficos**: Implementación personalizada de gráficos de tortugas (turtle)
- **Diseño**: Interfaz minimalista y responsiva con la fuente Outfit
